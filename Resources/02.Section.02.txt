• Adding Newtonsoft.Json Package
    from

    1. .NET CLI
        --> || > dotnet add package Newtonsoft.Json --version 13.0.3
    2. Package Manager
        --> || PM> NuGet\Install-Package Newtonsoft.Json -Version 13.0.3
    3. Package Reference
        --> || <PackageReference Include="Newtonsoft.Json" Version="13.0.3" />
    4. Paket CLI
        --> || > paket add Newtonsoft.Json --version 13.0.3
    5. Script & Interactive
        --> || > #r "nuget: Newtonsoft.Json, 13.0.3"
    6. Cake
        --> || // Install Newtonsoft.Json as a Cake Addin
                #addin nuget:?package=Newtonsoft.Json&version=13.0.3

        --> || // Install Newtonsoft.Json as a Cake Tool
                #tool nuget:?package=Newtonsoft.Json&version=13.0.3
